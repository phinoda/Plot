Privacy Policy

Last updated: May 15, 2026

Plot is local-first. The extension uses browser storage to keep your project workspace on your device. Plot does not collect, sell, share, or transmit your project data to a remote server.

Product

Plot is a Chrome extension by NoStock Studio that replaces the Chrome new tab page with a private workspace for planning, tracking, and reviewing personal projects.

Plot's single purpose is to help you manage projects, tickets, decisions, learnings, delivery progress, and lightweight project analytics from your new tab page.

Data Handled Locally

Plot stores the workspace data you create in the extension. This may include:

  • Projects and project names.
  • Tickets, tasks, delivered work, decisions, and learnings.
  • Notes, tags, timestamps, and project activity used for local analytics.
  • Layout, view, theme, date, and workspace preferences.
  • Optional local backup settings.

This data is stored locally in your browser using chrome.storage.local. If you choose a backup folder, Plot may also store a browser-managed folder permission locally in IndexedDB so it can continue writing your backup file.

Collection And Sharing

Plot does not collect your workspace data on NoStock Studio servers. Plot does not transmit your projects, tickets, notes, decisions, learnings, analytics, or backup files to a remote server.

Plot does not sell your data. Plot does not share your data with third parties. Plot does not use your data for advertising, credit decisions, data brokerage, or personalized ads.

Plot does not include third-party analytics, tracking pixels, or advertising SDKs.

Optional Backup

Plot includes an optional local backup feature. If you choose a folder, Plot writes a plot-backup.json file to that folder on your own device using the browser's File System Access API.

The backup file is local. Plot does not upload the backup file to NoStock Studio or to any third-party service.

Chrome Permissions

Plot requests the Chrome storage permission so it can save your projects, entries, layout, and preferences in your browser.

The production extension does not request host permissions and does not read or change the websites you visit.

Remote Code

Plot does not execute remote code. The extension runs code that is packaged with the extension.

User Control

You can delete projects, tickets, and entries inside Plot. You can also remove the extension or clear Chrome extension storage to remove locally stored Plot data from your browser profile.

If you created a local backup file, you can delete that file from the folder you selected.

Limited Use

The use of information received from Google APIs will adhere to the Chrome Web Store User Data Policy, including the Limited Use requirements.

Changes

If a future version of Plot adds optional sync, account, AI, analytics, or cloud features, this policy will be updated before those features are released.
